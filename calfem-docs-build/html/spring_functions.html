    spring1e¶

    Purpose:

    Compute element stiffness matrix for a spring element.

    Syntax:
    Ke = spring1e(ep)
    Description:

    \(\mathtt{spring1e}\) provides the element stiffness matrix \(\mathtt{Ke}\) for a spring element.

    The input variable

    \(\mathtt{ep} = [\,k\,]\)

    supplies the spring stiffness \(k\) or the analog quantity defined in Table Analogous quantities.

    Theory:

    The element stiffness matrix \(\mathbf{K}^e\), stored in \(\mathtt{Ke}\), is computed according to

    \[\begin{split}\mathbf{K}^e = \begin{bmatrix} k & -k \\ -k & k \end{bmatrix}\end{split}\]

    where \(k\) is defined by \(\mathtt{ep}\).

    spring1s¶

    Purpose:

    Compute spring force in a spring element.

    Syntax:
    es = spring1s(ep, ed)
    Description:

    \(\mathrm{spring1s}\) computes the spring force \(\mathrm{es}\) in a spring element.

    The input variable \(\mathrm{ep}\) is defined in \(\mathrm{spring1e}\) and the element nodal displacements \(\mathrm{ed}\) are obtained by the function \(\mathrm{extract}\).

    The output variable

    \[\mathrm{es} = \left[\,N\,\right]\]

    contains the spring force \(N\), or the analog quantity.

    Theory:

    The spring force \(N\), or analog quantity, is computed according to

    \[N = k \left(u_2 - u_1\right)\]
